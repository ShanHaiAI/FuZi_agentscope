# 夫子 FuZi
> ”师者，所以传道受业解惑也。——韩愈“  

## 项目简介
**夫子**（**FuZi**）是一个基于muti-agent实现的AI学习助手，旨在帮助用户完成全流程的技能自学。  
夫子的主要功能包括：学习路线规划、知识学习和讲解、自测考试和评分以及针对性强化等内容，基于agentic workflow，面向流程开发，通过muti-agent实现多AI agent交互。
